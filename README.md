#BackEnd part
Full stack web application for the preservation and diffusion of indigenous 
languages from Baja California Sur, Mexico. Built with Angular 18 and 
ASP.NET Core 9.0, featuring JWT authentication with role-based authorization, 
Cloudinary image storage, and a searchable dictionary of 635 vocabulary entries 
from the Pericú, Guaycura, and Cochimí peoples. Includes unit tests with xUnit 
and Moq covering authentication and authorization paths.

Tech stack: C# · ASP.NET Core 9.0 · Entity Framework Core · SQL Server · 
Angular 18 · TypeScript · Bootstrap 5 · Cloudinary · xUnit · Moq
